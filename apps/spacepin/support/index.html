1. Overview

SpacePin is a menu bar app for macOS that shows notes and images in small pinned windows.

2. Basic Usage

  • Open SpacePin from the menu bar.
  • Select New Note Pin to add a note.
  • Select Import Image... to add an image pin.
  • Select Open Manager to review and change detailed settings.

3. Main Features

  • Multiple pins
  • Move and resize
  • Lock
  • Click-through
  • Opacity adjustment
  • Note color selection
  • Font size adjustment
  • Restore pins after restarting the app

4. FAQ

Why does the app not appear in the Dock?

This is expected behavior. SpacePin is designed as a menu bar app.

Will pins always stay visible on full-screen Spaces?

Not completely. Because of macOS behavior and restrictions, display on full-screen Spaces cannot be fully guaranteed in every case.

What should I do if I turned on click-through and cannot click a pin?

Open the Manager or use the menu bar item to turn click-through off.
