1. 应用简介

SpacePin 是一款 macOS 菜单栏应用，可将笔记和图片显示在小型固定窗口中。

2. 基本用法

  • 从菜单栏打开 SpacePin。
  • 选择 New Note Pin 添加笔记。
  • 选择 Import Image... 添加图片固定项。
  • 选择 Open Manager 查看并更改详细设置。

3. 主要功能

  • 多个固定项
  • 移动和调整大小
  • 锁定
  • 点击穿透
  • 透明度调整
  • 选择笔记颜色
  • 调整字体大小
  • 应用重启后恢复固定项

4. 常见问题

为什么应用不会出现在 Dock 中？

这是正常行为。SpacePin 设计为菜单栏应用。

固定项在全屏 Space 中会始终显示吗？

无法完全保证。由于 macOS 的行为和限制，在全屏 Space 中的显示效果可能会因情况而异。

如果我开启了点击穿透，导致无法点击固定项，该怎么办？

请打开 Manager，或使用菜单栏项目关闭点击穿透。
