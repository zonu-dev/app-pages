1. 앱 소개

SpacePin은 메모와 이미지를 작은 고정 창으로 표시할 수 있는 macOS용 메뉴 막대 앱입니다.

2. 기본 사용법

  • 메뉴 막대에서 SpacePin을 엽니다.
  • New Note Pin으로 메모를 추가합니다.
  • Import Image...로 이미지 핀을 추가합니다.
  • Open Manager에서 상세 설정을 확인하고 변경할 수 있습니다.

3. 주요 기능

  • 여러 개의 핀
  • 이동 및 크기 조절
  • 잠금
  • 클릭 통과
  • 불투명도 조절
  • 노트 색상 변경
  • 글꼴 크기 조절
  • 앱 재시작 후 핀 복원

4. 자주 묻는 질문

왜 앱이 Dock에 표시되지 않나요?

정상 동작입니다. SpacePin은 메뉴 막대 앱으로 설계되었습니다.

핀은 전체 화면 Space에서도 항상 표시되나요?

완전히 보장되지는 않습니다. macOS의 동작과 제약 때문에 전체 화면 Space에서의 표시는 상황에 따라 달라질 수 있습니다.

클릭 통과를 켠 뒤 핀을 클릭할 수 없게 되었습니다. 어떻게 해야 하나요?

Manager를 열거나 메뉴 막대 항목을 사용해 클릭 통과를 꺼 주세요.
