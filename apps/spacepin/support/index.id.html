1. Ringkasan

SpacePin adalah aplikasi menu bar untuk macOS yang menampilkan catatan dan gambar dalam jendela kecil yang dipin.

2. Cara penggunaan dasar

  • Buka SpacePin dari menu bar.
  • Pilih New Note Pin untuk menambahkan catatan.
  • Pilih Import Image... untuk menambahkan pin gambar.
  • Pilih Open Manager untuk meninjau dan mengubah pengaturan detail.

3. Fitur utama

  • Banyak pin
  • Pindahkan dan ubah ukuran
  • Kunci
  • Click-through
  • Penyesuaian opasitas
  • Pemilihan warna catatan
  • Penyesuaian ukuran font
  • Memulihkan pin setelah aplikasi dimulai ulang

4. FAQ

Mengapa aplikasi tidak muncul di Dock?

Ini adalah perilaku yang diharapkan. SpacePin dirancang sebagai aplikasi menu bar.

Apakah pin akan selalu terlihat di Space layar penuh?

Tidak sepenuhnya. Karena perilaku dan batasan macOS, tampilan pada Space layar penuh tidak dapat dijamin dalam setiap kasus.

Apa yang harus saya lakukan jika saya menyalakan click-through dan tidak bisa mengklik pin?

Buka Manager atau gunakan item menu bar untuk mematikan click-through.
