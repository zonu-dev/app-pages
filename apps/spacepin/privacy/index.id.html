1. Pendahuluan

Aplikasi "SpacePin" ("Aplikasi") menghormati privasi Anda dan berkomitmen untuk melindungi informasi Anda. Kebijakan Privasi ini menjelaskan informasi apa yang ditangani Aplikasi dan bagaimana informasi tersebut digunakan.

2. Informasi yang Kami Kumpulkan dan Simpan

Aplikasi ini adalah aplikasi macOS yang menampilkan catatan dan gambar dalam jendela kecil yang dipin. Aplikasi tidak memerlukan pendaftaran akun dan tidak mengumpulkan informasi pribadi, data analitik, pengenal iklan, atau informasi lokasi.

Aplikasi menyimpan data secara lokal di perangkat Anda, termasuk:

  • Isi dan judul catatan
  • Posisi, ukuran, dan opasitas jendela
  • Warna catatan dan ukuran font
  • Status kunci, status click-through, dan status terlipat
  • Pengaturan lain yang diperlukan untuk memulihkan catatan dan jendela Anda

Aplikasi hanya menangani gambar yang Anda pilih untuk dimuat. Gambar yang dimuat akan disalin ke area penyimpanan lokal Aplikasi agar dapat dipulihkan setelah Aplikasi dimulai ulang.

3. Cara Kami Menggunakan Informasi

Informasi yang disimpan secara lokal hanya digunakan untuk tujuan berikut:

  • Menampilkan catatan dan gambar Anda dalam jendela yang dipin
  • Memulihkan catatan, gambar, dan status jendela setelah Aplikasi dimulai ulang
  • Mempertahankan pengaturan tampilan dan perilaku yang Anda pilih

4. Layanan Pihak Ketiga

Aplikasi tidak menggunakan SDK iklan, SDK analitik, atau pelacak pihak ketiga.

Aplikasi tidak mengunggah catatan atau gambar Anda ke server kami maupun ke server pihak ketiga.

5. Berbagi dengan Pihak Ketiga

Kami tidak memberikan informasi, catatan, atau gambar Anda kepada pihak ketiga kecuali diwajibkan oleh hukum.

6. Penyimpanan dan Penghapusan Data

Data yang disimpan oleh Aplikasi berada secara lokal di perangkat Anda. Data tersebut akan tetap ada sampai Anda menghapusnya di dalam Aplikasi atau mencopot pemasangan Aplikasi.

Gambar yang disalin ke penyimpanan lokal Aplikasi untuk pemulihan juga hanya disimpan secara lokal dan tidak diunggah ke server mana pun.

7. Keamanan

Aplikasi menyimpan catatan, gambar, dan pengaturan terkait secara lokal di perangkat Anda. Karena Aplikasi tidak menjalankan server backend untuk data ini, konten Anda tidak dikirimkan kepada kami dalam penggunaan normal Aplikasi.

8. Privasi Anak

Aplikasi tidak dengan sengaja mengumpulkan informasi pribadi dari anak-anak di bawah usia 13 tahun.

9. Perubahan pada Kebijakan Ini

Kami dapat memperbarui Kebijakan Privasi ini jika diperlukan. Kebijakan yang diperbarui akan berlaku saat dipublikasikan di halaman ini.
